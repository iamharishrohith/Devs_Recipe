<p>Follow these steps to set up the project locally.</p>

<h3>Prerequisites</h3>

<p>Ensure you have <strong>Node.js</strong> (v18 or higher) installed on your machine.</p>

<h3>Installation</h3>

<ol>
  <li>
    <p><strong>Clone the repository</strong></p>
    <pre><code>git clone https://github.com/iamharishrohith/Devs_Recipe.git
cd Devs_Recipe</code></pre>
  </li>
  <li>
    <p><strong>Install dependencies</strong></p>
    <pre><code>npm install
# or
pnpm install
# or
yarn install</code></pre>
  </li>
  <li>
    <p><strong>Run the development server</strong></p>
    <pre><code>npm run dev</code></pre>
  </li>
  <li>
    <p><strong>Open the app</strong><br>
    Visit <a href="http://localhost:3000">http://localhost:3000</a> in your browser.</p>
  </li>
</ol>
